# ChatBox

ChatBox is a lightweight WhatsApp-like web chat application built with **HTML, CSS, and JavaScript**. It is designed for educational purposes and can also be converted into an **Android APK** using WebView or online APK builders. This project helps beginners learn how to create chat interfaces and convert web apps to mobile apps.

## Features

- Mobile number login system (demo OTP)
- Friends / Chat List
- Profile page
- Dark Mode support
- Basic online chat simulation
- Fully front-end, easy to modify
- Can be converted to Android APK
